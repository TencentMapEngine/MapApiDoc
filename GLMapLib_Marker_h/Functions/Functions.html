MapMarkerSetMainMarker

void MapMarkerSetMainMarker(
    void *pWorld,
    int subMarkerID,
    int mainMarkerID);
Discussion

为当前marker(子marker)设置一个MainMarker(主marker)，添加一层虚拟的关联关系。mainMarkerID=-1，清除关系。 如果MainMarker被避让掉，当前marker也一同被避让掉； 如果MainMarker没有被避让掉，当前marker按照自身情况避让。 如果用户主动对MainMarker调用SetHidden操作，当前marker不受影响。
